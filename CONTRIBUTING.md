# Contributing

1. Create a Python 3.10 or later virtual environment.
2. Install development dependencies with `python -m pip install -e ".[dev]"`.
3. Add or update tests for every behavioral change.
4. Run `python -m pytest` and `ruff check .`.
5. Keep the public objective and directed KNN accounting documented.

Bug reports should include a minimal input, package versions, the random state,
and the complete traceback.
